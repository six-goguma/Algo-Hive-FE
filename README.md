## 🚩 주요 기능
- MARKDOWN 형식으로 작성 가능한 게시판: 게시판 생성, 보기, 수정, 삭제, 좋아요, 댓글 작성의 기능을 사용할 수 있다.
- 채팅: 채팅방을 생성할 수 있고, 다른 사용자와 채팅을 실시간으로 이용할 수 있다.
- AI 코드리뷰: 코딩 테스트 문제를 풀고 붙여 넣으면 ai로부터 코드리뷰를 받을 수 있다.
